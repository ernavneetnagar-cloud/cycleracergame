# Cycle Racer Game 🚴‍♂️

A thrilling Android game application where players race on cycles, built with TypeScript and powered by Google Gemini AI integration.

## About the Project

Cycle Racer Game is an interactive mobile game that brings the excitement of cycle racing to your fingertips. The game combines engaging gameplay mechanics with AI-powered features using Google Gemini API, creating an immersive gaming experience.

### Key Features

- 🎮 **Engaging Gameplay** - Experience fast-paced cycle racing action
- 🤖 **AI Integration** - Powered by Google Gemini API for enhanced gameplay
- 📱 **Mobile Optimized** - Designed for Android devices
- ⚡ **Built with TypeScript** - Modern, type-safe development

## Tech Stack

- **Language:** TypeScript
- **Platform:** Android
- **AI Integration:** Google Gemini API
- **Build Tool:** Node.js & npm

## Prerequisites

Before you begin, ensure you have the following installed:

- **Node.js** (v14 or higher)
- **npm** (comes with Node.js)
- **Gemini API Key** (from [Google AI Studio](https://aistudio.google.com))

## Installation

### 1. Clone the Repository

```bash
git clone https://github.com/ernavneetnagar/cycleracergame.git
cd cycleracergame
```

### 2. Install Dependencies

```bash
npm install
```

### 3. Configure Gemini API Key

Create or update the `.env.local` file in the project root and add your Gemini API key:

```env
GEMINI_API_KEY=your_api_key_here
```

You can get your API key from [Google AI Studio](https://aistudio.google.com).

### 4. Run the Application

```bash
npm run dev
```

The application will start in development mode and be accessible at `http://localhost:3000` (or as configured).
